Co znaczy skrót ID? Identyfikator
Jaki jest cel stosowania ID? W CSS oznacza ID selektora.
Jakie jest znaczenie znaku # Przed definiowaniem identyfikatora występuje znak #
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Użycie znaku # przed nazwą ID w CSS (np. #header { color: blue; }) identyfikuje i stylizuje element o danym ID.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  • ID powinno być unikalne w dokumencie.
  • Stosować tylko raz na stronę.
  • Używać dla kluczowych elementów, które muszą być jednoznacznie rozpoznawalne.
